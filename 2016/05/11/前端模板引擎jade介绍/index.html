前端模板引擎jade介绍

Posted on 2016-05-11   |  

jade是一种可以提高前端开发写代码效率的工具


名词解释：

  • Jade：是一个基于javascript，可以供node与浏览器环境使用的一个前端模板引擎。

jade特性：

  • 客户端支持
  • 代码高可读
  • 灵活的缩进
  • 块展开
  • Mixins
  • 静态包含
  • 属性改写
  • 安全，默认代码是转义的
  • 运行时和编译时上下文错误报告
  • 命令行下编译jade模板
  • HTML5 模式 (使用 !!! 5 文档类型)
  • 在内存中缓存(可选)
  • 合并动态和静态标签类
  • 可以通过 filters 修改树
  • 模板继承
  • 原生支持 Express JS
  • 通过 each 枚举对象、数组甚至是不能枚举的对象
  • 块注释
  • 没有前缀的标签
  • AST Filters
  • 过滤器

jade安装？

在终端中输入以下命令
npm install -g jade
此时系统出现报错，安装失败，如图显示：

安装解决方案：（jade作为模块中的开发依赖包，所以在命令前面输入sudo，代表安装权限，然后再输入电脑密码即可安装成功）
sudo npm install --save-dev gulp-jade
