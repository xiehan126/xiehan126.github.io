    • 名词解释：

    • Node：Node是一个服务器端 JavaScript 解释器。它本身运行Google V8 JavaScript引擎，所以速度和性能非常好。

    • GitHub：作为开源代码库以及版本控制系统，同时GitHub可以托管各种git。

    • Hexo是一个开源的静态博客生成器,同时它需要我们手动进行搭建。

    以下采用（mac系统为例）

    一、安装NodeJs终端

    • 在nodeJs官网下载它的终端，网址： https://nodejs.org/en/
    • 建议选择稳定版本下载。
    • 下载安装完毕以后，在终端输入node -v即可确认nodeJs是否安装成功， 同时会出现它的版本号。

    二、安装Git

    • 安装过Xcode可以直接跳过这步，因为Xcode自带Git。
    • 如果没有，那么Mac和Windows都可以直接到官网下载安装。

    三、安装hexo

    • 在终端界面下，输入npm install hexo 如果权限不对操作失败请提升权限。
    • 同样的，在终端页面输入hexo-v，出现它的版本号，或者输入hexo出现很多内容，也可以证明安装成功。

    以上三步安装成功后，可以在终端对博客进行基本设置了。

    1. mkdir blog:（关于文件名称可以任意起名字），即可创建一个文件夹，随后桌面会显示该文件夹。
    2. hexo init:接下来就是等待的时间了，你会发现会出现自动生成了很多文件，这是在初始化博客目录。

    3. hexo new title：创建新博客，可以给新博客起一个名字。创建成功后，如图所示，会在新创建的博客文件夹中source_posts的路径下面显示。

    4. 用Mou编辑完，就可以完成并发布了。
    5. hexo server，启动本地服务器业务，然后再浏览器地址栏输入http://0.0.0.0:4000/这样就可以预览你写的博客了。
